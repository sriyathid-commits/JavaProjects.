# JavaProjects Repository

Welcome to the **JavaProjects** repository!  
This collection showcases five beginner‑to‑intermediate Java applications designed to strengthen your programming fundamentals and demonstrate practical software development skills.

---

## 📘 Bug Tracking System
A simple Java application to log, track, and manage software bugs.  
**Features:**
- Add new bug reports  
- Assign bugs to developers  
- Track bug status (open, in progress, resolved)  
- Generate summary reports  

---

## 💬 Chatting Application
A Java‑based chat system for real‑time communication between users.  
**Features:**
- User authentication  
- Send and receive messages  
- Group chat functionality  
- Basic server‑client architecture  

---

## 👥 Customer Relationships Manager (CRM)
A Java application to manage customer data and interactions.  
**Features:**
- Store customer profiles  
- Track sales and communication history  
- Generate business reports  
- Search and filter customer records  

---

## 📧 Email System
A Java program to simulate sending and receiving emails.  
**Features:**
- Compose and send emails  
- Inbox and outbox management  
- Basic spam filtering  
- Simple user interface  

---

## 🎮 Advanced Sudoku Game
A Java‑based Sudoku puzzle game with advanced features.  
**Features:**
- Puzzle generation  
- Multiple difficulty levels  
- Solver and hints system  
- Interactive console or GUI play
